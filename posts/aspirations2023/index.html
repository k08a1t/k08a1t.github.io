Aspirations for 2023

2023-01-01 :: k08a1t
#Others 

昨年はやりたいことがふんわりした状態で過ごしていたので、今年は抱負を考えてみました。

自分の活動を整理することで自身の能力を把握し、即戦力として働けるように準備する⌗

あと一年と少しすれば社会人なので、その準備をする年になるかな…。というところです。

具体的には、ブログやGitHabで自分の学習したことや参加したイベントなどについてまとめつつ、振り返っては反省/改善していこうと思っています。 　　

細かい目標も決めておきます

  • 情報処理安全確保支援士取得
  • セキュリティイベントへの参加(SecurityCamp, SecHack365, CODE BLUE…)
  • 英語能力を上げる(主に語彙、会話)

こんなところでしょうか。 まずは就活を頑張らなければいけないのですがそれを理由に技術力が衰えていくのは嫌なので、
体調管理には気をつけつつも全力で突っ走る一年にしたいです。
